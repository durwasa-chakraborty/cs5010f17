Example of Compiler Optimization

Consider the following method, which is written in Java: 

      // Returns the inner product of the given arrays.
      
      static double ip (double[] a, double[] b) {
          int n = a.length;
          double result = 0.0;
          for (int i = 0; i < n; i = i + 1)
              result = result + a[i] * b[i];
          return result;
      }

If you didn't know your Java compiler is an optimizing compiler, you might worry so much about the micro-efficiency of that method that you decide to write uglier code in an uglier programming language: 

      double ip (register double *a, register double *b, register int n) {
        register double result = 0.0;
        while (n-- > 0) {
          result += *(a++) * *(b++);
        }
        return result;
      }
