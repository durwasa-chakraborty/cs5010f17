What Do You Gain By Writing Ugly Code?

I wrote a benchmark that calls ip one million times on arrays of length 1000. I ran both the Java and C versions of that benchmark on a Linux machine.

The Java version of the benchmark printed its result almost instantly. Its running time was dominated by the time required to load and run the JVM.

When I used gcc to compile the ugly code, without optimization, the computation took several seconds.

When I used gcc -O4 to compile the ugly code, the computation took too little time to measure.
