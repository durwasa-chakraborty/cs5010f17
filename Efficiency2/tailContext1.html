Tail Contexts in Java

Within a Java method whose return type is not void, the expressions that appear in a tail context are defined as follows.

  • E is in a tail context if it appears in a return statement of the form 
              return E;
  • If a conditional expression is in a tail context, its then and else expressions (but not its test expression) are both in a tail context.

Any expression that is not in tail context according to those rules is not in a tail context.

Intuition: An expression is in tail context if and only if, when it is evaluated, its value will be returned immediately by the method in which the expression appears.
