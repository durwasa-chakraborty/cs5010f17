Learning Objectives

By the end of this lesson, you should be able to

  • distinguish non-tail recursion from tail recursion
  • explain the main differences between proper tail recursion, tail call optimization, and improper tail recursion
  • understand why a program may suffer from stack overflows when a faithful translation of the program into some other programming language never overflows the stack
  • rewrite straightforward self-tail-recursions into while or for loops
