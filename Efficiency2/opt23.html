Optimizations that Improve Asymptotic Efficiency

Compiler optimization can also recognize when a functional update to an array can be performed by side effect instead of creating a new array that differs from the original in only one place.

For a one-dimensional array of length n, that optimization reduces the cost of the update from Θ(n) to O(1).

The resources for Module 12 include a paper that proves the correctness of that optimization.
