Abstract Base Class for SortedIMap<K,V>

We are almost ready to implement binary search trees.

Our implementation of binary search trees will implement the SortedIMap<K,V> interface.

After we implement binary search trees, we might write another implementation of SortedIMap<K,V>. For example, we might decide to implement red-black trees.

Anticipating multiple implementations of the SortedIMap<K,V> interface, we will define an abstract base class for implementations of that interface, just as we defined an abstract base class for implementations of the IMap<K,V> interface.

In fact, our SortedIMapBase<K,V> class will extend the IMapBase<K,V> class. In Java, it is not unusual for one abstract base class to extend a more general abstract base class.
