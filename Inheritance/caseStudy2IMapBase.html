The IMapBase<K,V> Class

Before we implement binary search trees, we will design an abstract base class that can be used by our implementation of binary search trees and also by our implementation that uses linear search.

To test that abstract base class, we will rewrite the IMapList<K,V> class to inherit from it.

We haven't yet written the IMapBST class, and don't want to write it until we have written and tested the new base class. During that testing, we will temporarily change the static factory method that takes a comparator to ignore the comparator: 

          @SuppressWarnings("unchecked")
          public static <K,V> SortedIMap<K,V> empty (Comparator<? super K> c) {
              // FIXME: IMapBST isn't defined yet
              // return IMapBST.empty(c);
              return empty();
          }

Flagging the temporary patch with a FIXME or TODO comment helps programmers to remember to repair it later.
