Summary

  • Two predefined abstract base classes, AbstractMap and AbstractSet, made it much easier to implement immutable maps that support the operations of the predefined Map<K,V> interface in addition to the basic operations needed for immutable maps.
  • Defining an abstract base class for all immutable maps made it much easier to define multiple implementations of the IMap<K,V> interface.
  • Defining an abstract base class for binary search trees made it easier to define classes that represent empty and non-empty trees.

In short, inheritance made our implementations of these abstract data types a lot easier and simpler.
