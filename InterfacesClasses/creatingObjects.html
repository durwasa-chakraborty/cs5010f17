Here is the Java syntax for creating objects: 

          new Class1 (10, 20, 14)
          new Class1 (35, 15, 5)
          new Class3 (15, 35, 5)

Those are expressions, so they have to be placed into some kind of Java context where an expression can go.

Here is the Java syntax for defining variables whose values are given by the three expressions above: 

          Interface1 obj1 = new Class1 (10, 20, 14);
          Interface1 obj2 = new Class1 (35, 15, 5);
          Interface1 obj3 = new Class3 (15, 35, 5);

Testing is more complicated with objects, partly because we have to tell Java when two objects should be considered equal. We will discuss testing of objects, classes, and interfaces in another lesson.
