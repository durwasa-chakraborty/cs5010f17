Design #2: Pulling Information

In our second design, we add an intersects method to the interface. 

      // A Ball2 is an object of any class that implements Ball2.
      
      interface Ball2 {
          // RETURNS: x or y coordinates or radius of this ball,
          //     in pixels
      
          int getX ();
          int getY ();
          int getR ();
      
          // Does the given ball intersect with this one?
      
          boolean intersects (Ball2 b);
      }
